# BMI535slides
 
Slides/Jupyter Notebooks for BMI 535/635 Processing of Large Scale Data

# Required to Run

Installation instructions are in `00_load_database.ipynb`.

- Anaconda/Jupyter
- postgresql (installation instructions are here)
- ipython-sql
- psycopg2
- pgspecial
